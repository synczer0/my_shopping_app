# Simple Shopping List

A simple shopping list app

## Features
1. Implemented with CRUD
2. Using Provider

## Usage
```
1. git clone https://github.com/synczer0/my_shopping_app.git
2. cd /<your_directory>/my_shopping_app
3. flutter clean
4. flutter pub get
5. flutter run
```
